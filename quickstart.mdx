This guide will help you set up CoreDump and start tracking your coding activity as quickly as possible. You'll create an account, install the VS Code extension, and see your first tracked activity.

<Note>
  This quickstart is for using the hosted version of CoreDump. If you want to self-host, see the [Installation Guide](/installation).
</Note>

## Get started in 3 steps

<Steps>
  <Step title="Create your CoreDump account">
    Visit [coredump.vercel.app](https://coredump.vercel.app) and sign up for a free account.

    You can sign up using:
    - **Google authentication** (recommended for fastest setup)
    - **Email and password** registration

    After signing up, you'll be redirected to your dashboard.
  </Step>

  <Step title="Get your private key">
    Your private key is what connects the VS Code extension to your CoreDump account.

    To find your private key:
    1. Navigate to your dashboard after logging in
    2. Look for the **Private Key** section
    3. Click to reveal and copy your private key

    <Warning>
      Keep your private key secure. Anyone with your private key can track activity to your account.
    </Warning>

    Save this key somewhere safe - you'll need it in the next step.
  </Step>

  <Step title="Install and configure the VS Code extension">
    Now let's install the CoreDump extension and connect it to your account.

    ### Install the extension

    You can install the extension in two ways:

    **Option 1: From VS Code Marketplace (recommended)**
    1. Open Visual Studio Code
    2. Click the Extensions icon in the Activity Bar (or press `Ctrl+Shift+X` / `Cmd+Shift+X`)
    3. Search for "CoreDump"
    4. Click **Install** on the extension by Unsafezero

    **Option 2: Direct download**
    
    Visit the [VS Code Marketplace](https://marketplace.visualstudio.com/items?itemName=Unsafezero.coredump) and click **Install**.

    ### Configure your private key

    Once installed, connect the extension to your account:

    1. Open the Command Palette (`Ctrl+Shift+P` / `Cmd+Shift+P`)
    2. Type and select **CoreDump: Enter Private Key**
    3. Paste your private key from Step 2
    4. Press Enter to save

    <Info>
      You may need to restart VS Code for the extension to start tracking activity.
    </Info>
  </Step>
</Steps>

## Start coding and see your stats

That's it! You're all set up. Now just start coding in VS Code and CoreDump will automatically track your activity.

The extension tracks:
- **Coding time** per language
- **Active files** you're working on
- **Time spent** in each coding session

Activity is sent to CoreDump every few minutes while you're actively coding.

### View your dashboard

To see your tracked activity:

1. Go to [coredump.vercel.app/dashboard](https://coredump.vercel.app/dashboard)
2. You'll see your coding stats including:
   - Total coding time
   - Current streak
   - Language breakdown
   - Daily and weekly activity charts

<Info>
  It may take a few minutes for your first activity to appear on the dashboard after you start coding.
</Info>

## What you'll see

After coding for a bit, your dashboard will display:

**Activity metrics**
- Today's coding time
- 7-day coding time
- 30-day coding time
- Total all-time coding time

**Streak information**
- Current daily streak
- Maximum streak achieved
- Streak status (active/at risk)

**Language breakdown**
- Visual charts showing time per language
- Language badges based on proficiency
- Detailed language statistics

**Achievements**
- Points earned (1 point per 10 minutes of coding)
- Unlocked achievement badges
- Progress toward next achievements

## Join the leaderboard

Once you've logged some coding time, you can compete with other developers:

1. Visit [coredump.vercel.app/leaderboard](https://coredump.vercel.app/leaderboard)
2. View rankings based on:
   - Last 24 hours coding time
   - Last 7 days coding time
3. Click on any developer's profile to see their stats

## Troubleshooting

### Extension not tracking

If your coding activity isn't being tracked:

1. Verify your private key is entered correctly
2. Restart VS Code
3. Check that the extension is enabled
4. Make sure you're actively editing files (not just viewing)

### Activity not showing on dashboard

- Activity updates may take a few minutes to appear
- Refresh your dashboard page
- Ensure the extension shows as active in VS Code's status bar

### Need more help?

Visit the [GitHub repository](https://github.com/unsafe0x0/CoreDump) to report issues or check the troubleshooting section above for common problems.
