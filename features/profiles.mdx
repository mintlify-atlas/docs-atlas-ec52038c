Every CoreDump user has a public developer profile that showcases their coding statistics, achievements, and activity patterns. Profiles help you track your own progress and discover other developers in the community.

## Your Profile Overview

Your developer profile displays:

- **Profile header** with your name and social links
- **Weekly activity chart** showing this week's coding pattern
- **Statistics grid** with 8 key metrics
- **Achievements showcase** with all unlocked badges
- **Language breakdown** with detailed time tracking

<Info>
Your profile is automatically public and accessible at `/developer/[your-username]`.
</Info>

## Profile Header

The profile header contains your identity and social connections:

### Display Information
- **Full name** - Your display name on CoreDump
- **Profile image** - Avatar from your linked GitHub account
- **GitHub username** - Clickable link to your GitHub profile
- **Twitter username** - Clickable link to your Twitter/X profile
- **Personal website** - Optional link to your portfolio or blog

<Tip>
Update your social links and website URL from your dashboard settings to build your developer brand.
</Tip>

## Weekly Coding Activity

Profiles feature a bar chart showing coding time for each day of the current week:

- **7-day view** - Monday through Sunday
- **Hours and minutes** - Precise time for each day
- **Visual comparison** - Quickly identify your most productive days
- **Current week only** - Resets every Monday

### Reading the Chart

The weekly chart helps you:
- Identify your most productive days
- Spot patterns in your coding schedule
- Share your consistency with others
- Track week-over-week improvements

<Note>
The weekly chart displays the same data on your dashboard and public profile. There's no separate private view.
</Note>

## Profile Statistics

Eight key metrics appear in your stats grid:

### Current Streak
Consecutive days of coding activity. This publicly demonstrates your coding consistency and discipline.

### Max Streak
Your all-time record for consecutive days. This never decreases and shows your peak performance.

### Total Time
Cumulative coding time across all languages since joining CoreDump. Displayed as hours and minutes.

### Language Count
Number of different programming languages you've used. Shows your versatility as a developer.

### Top Language
Your most-used programming language by total time. Updates automatically as your activity changes.

### This Month
Total coding time in the current calendar month. Resets on the first of each month.

### Achievements
Number of achievements unlocked out of total available. Shows your gamification progress.

### Dump Points
Total points earned from coding activity and achievements. Used for leaderboard rankings.

<Info>
All statistics update in real-time as you code with the CoreDump editor plugin installed.
</Info>

## Achievements Display

Your profile prominently features all achievements you've unlocked:

- **Visual badges** - Icon for each achievement
- **Achievement name** - Title like "7-Day Streak" or "Night Owl"
- **Description** - How the achievement was earned
- **Points value** - How many Dump Points it awarded

Unlocked achievements display in full color, while locked achievements may appear grayed out or hidden depending on the view.

<Tip>
Achievements are a great conversation starter. Other developers can see what you've accomplished and get inspired to earn the same badges.
</Tip>

## Language Statistics

The language breakdown section shows detailed metrics for every programming language you've used:

**For each language:**
- Language name and icon
- Total coding time
- Percentage of overall activity
- Last 24 hours activity
- Last 7 days activity
- Last 30 days activity

**Sorting:**
Languages appear sorted by total time, with your most-used languages at the top.

<Note>
Language detection is automatic based on file extensions. The CoreDump plugin identifies languages as you work in different files.
</Note>

## Public vs Private Information

### What's Public

Everyone can see on your profile:
- Your name and profile image
- Social media links (GitHub, Twitter, website)
- All coding statistics and metrics
- Weekly activity chart
- Achievements you've unlocked
- Language breakdown and time stats
- Current and max streak

### What's Private

These items never appear on your public profile:
- Your email address
- Your private API key
- Account settings and preferences
- Exact timestamps of coding sessions
- Code content or project names
- Specific file paths or repositories

<Info>
CoreDump tracks coding time and language usage only. Your actual code and project details remain completely private.
</Info>

## Customizing Your Profile

To update your profile information:

1. Navigate to your dashboard
2. Click the settings icon (gear icon in top right)
3. Update any of these fields:
   - Full name
   - GitHub username
   - Twitter username
   - Personal website URL
4. Save changes

Your profile updates immediately across the platform.

<Tip>
Add a personal website URL to drive traffic to your portfolio. Many developers use their CoreDump profile as part of their online presence.
</Tip>

## Viewing Other Profiles

To view another developer's profile:

**From Leaderboard:**
- Click any developer's name on the leaderboard
- You'll navigate to their public profile

**Direct URL:**
- Visit `/developer/[their-username]`
- Replace `[their-username]` with their actual username

**From Profile:**
- No built-in "follow" or "friends" system currently
- Bookmark profiles manually to track other developers

<Note>
All registered CoreDump users have public profiles. You cannot make your profile completely private, but you control what social links appear.
</Note>

## Profile URL Structure

Your profile URL follows this pattern:
```
https://coredump.app/developer/your-username
```

Where `your-username` is your GitHub username used during registration.

**Sharing Your Profile:**
- Include it in your resume or portfolio
- Share on social media to showcase consistency
- Link from your GitHub README
- Add to your Twitter bio

<Tip>
Your CoreDump profile serves as proof of your coding consistency and habits - valuable for job applications and networking.
</Tip>

## Profile Performance

Profiles load quickly thanks to:
- Cached statistics data
- Optimized database queries
- Efficient chart rendering
- Lazy-loaded achievement images

<Info>
If a profile shows a loading state, it's fetching the latest statistics from the database. This ensures you always see current data.
</Info>

## Understanding Profile Context

When viewing other profiles, remember:

**Streaks depend on consistency** - A 30-day streak shows incredible discipline, regardless of daily hours.

**Total time accumulates** - Someone with 500 hours has been using CoreDump longer or more intensively than someone with 50 hours.

**Language counts vary by role** - Backend developers might use fewer languages than full-stack developers.

**This month resets** - High "This Month" totals early in the month are particularly impressive.

<Note>
Don't compare yourself too harshly to others. Everyone has different schedules, roles, and coding patterns. Use profiles for inspiration, not stress.
</Note>
