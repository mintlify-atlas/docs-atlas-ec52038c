The Dashboard is your central hub for tracking and analyzing your coding activity. It provides a comprehensive overview of your coding metrics, streaks, achievements, and language usage.

## Overview

When you log into CoreDump, your dashboard displays:

- **Personal greeting** with your name
- **Profile information** including GitHub and Twitter links
- **Your private API key** for editor integration
- **Weekly activity chart** showing daily coding time
- **Statistics grid** with key metrics
- **Achievements** you've unlocked
- **Language breakdown** of your coding activity

<Info>
Your dashboard updates in real-time as you code with the CoreDump editor plugin installed.
</Info>

## Weekly Activity Chart

The activity chart at the top of your dashboard visualizes your coding time across the current week (Monday through Sunday). Each bar represents one day, showing the total hours and minutes you spent coding.

**How to read the chart:**
- Hover over any bar to see exact coding time for that day
- The chart resets every Monday with a new week
- Zero-activity days still appear on the chart for tracking purposes

<Tip>
Aim for consistent daily activity to maintain your coding streak and climb the leaderboards.
</Tip>

## Statistics Overview

Your dashboard displays eight key metrics in a grid format:

### Current Streak
The number of consecutive days you've coded. This counter increases when you log coding activity each day and resets to zero if you skip a day.

### Max Streak
Your personal record for consecutive days of coding. This value never decreases - it tracks your best performance over time.

### Total Time
The cumulative coding time across all languages and projects since you joined CoreDump. Displayed in hours and minutes format.

### Language Count
The total number of different programming languages you've used. Each unique language in your activity history adds to this count.

### Top Language
Your most-used programming language based on total coding time. This updates automatically as your activity changes.

### This Month
Total coding time for the current calendar month. Resets to zero on the first day of each month.

### Achievements
The number of achievements you've unlocked out of the total available. Click to view your full achievement collection.

### Dump Points
Your total points earned from coding activity and achievements. Points contribute to your leaderboard ranking.

<Note>
All time-based metrics update automatically as you code with the editor plugin active.
</Note>

## Language Breakdown

The language section shows detailed statistics for each programming language you've used:

- **Language name** with icon
- **Total time** spent in that language
- **Percentage** of your overall coding time
- **Last 24 hours** activity
- **Last 7 days** activity
- **Last 30 days** activity

Languages are sorted by total time, with your most-used languages appearing first.

## Private API Key

Your dashboard prominently displays your private API key, which you need to connect the CoreDump editor plugin:

1. Copy your private key from the dashboard
2. Install the CoreDump plugin for your code editor
3. Paste the key into the plugin settings
4. Start coding - your activity will automatically track

<Info>
Keep your private key secure. Anyone with this key can track coding activity to your account.
</Info>

## Profile Actions

From your dashboard header, you can:

- **Return home** via the home icon
- **Toggle dark/light theme** for comfortable viewing
- **Access settings** to update your profile information

## Accessing Your Dashboard

To view your dashboard:

1. Sign in to your CoreDump account
2. You'll automatically land on your dashboard
3. Or navigate to `/dashboard` from anywhere on the site

<Tip>
Bookmark your dashboard URL for quick access to your coding stats throughout the day.
</Tip>

## Understanding Your Data

Your dashboard data comes from the CoreDump editor plugin, which tracks:

- **Active coding time** (not idle time)
- **Language detection** based on file extensions
- **Real-time synchronization** with the CoreDump servers
- **Privacy-first tracking** (only language and time, not code content)

<Note>
The dashboard only displays data when you're actively typing or editing code. Idle time in your editor doesn't count toward your statistics.
</Note>
