## Endpoint

```
GET /api/dashboard
```

## Description

Retrieves comprehensive dashboard data for the authenticated user, including:

- User profile information
- Private key (visible only to the user)
- Coding statistics and streaks
- Activity breakdown by language
- Daily activity for the current week
- Monthly activity history
- Achievements

## Authentication

Requires an active session via NextAuth. The user must be logged in.

## Request Parameters

No parameters required. User identification is based on the session.

## Error Responses

### Unauthorized (401)

```json
{
  "message": "Unauthorized",
  "status": 401
}
```

Returned when no valid session exists or user is not logged in.

### User Not Found (404)

```json
{
  "message": "User not found",
  "status": 404
}
```

Returned when the authenticated session references a user that no longer exists.

### Internal Server Error (500)

```json
{
  "message": "Something went wrong",
  "status": 500
}
```

Returned when an unexpected error occurs.

## Example Request

```bash
curl -X GET https://your-domain.com/api/dashboard \
  -H "Cookie: next-auth.session-token=your-session-token"
```

<Note>
  This endpoint requires browser-based authentication via NextAuth. You'll need to be logged in through the web interface to access this data.
</Note>

## Daily Activity Week Range

The `dailyActivity` array always contains 7 entries representing Monday through Sunday of the current week (week starts on Monday). Days without activity will have a duration of 0.
