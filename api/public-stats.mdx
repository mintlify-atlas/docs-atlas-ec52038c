<Note>
  The `/api/public-stats` endpoint does not exist as a separate route.
  
  For public coding statistics, please use the **[GET /api/stats](/api/stats)** endpoint instead.
</Note>

## What You're Looking For

If you need public coding statistics for a user, the `/api/stats` endpoint provides:

- Total coding time today (last 24 hours)
- Total coding time this week (last 7 days)
- Formatted as human-readable strings (e.g., "8h 45m")
- CORS-enabled for cross-origin requests
- No authentication required

## Quick Example

```bash
curl -X GET 'https://your-domain.com/api/stats?username=johndoe'
```

```json
{
  "username": "johndoe",
  "weekTotal": "12h 30m",
  "todayTotal": "2h 15m"
}
```
