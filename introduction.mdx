## Welcome to CoreDump

CoreDump is a comprehensive coding tracking platform that helps developers monitor their progress, maintain daily streaks, compete on leaderboards, and enhance their programming skills through gamification.

Whether you're building consistent coding habits, competing with other developers, or showcasing your achievements, CoreDump provides the tools and insights you need to level up your development journey.

## How it works

CoreDump seamlessly integrates with your development workflow in three simple steps:

<Steps>
  <Step title="Install the VS Code extension">
    Download the CoreDump extension from the Visual Studio Code marketplace. The extension runs silently in the background while you code.
  </Step>
  
  <Step title="Connect your account">
    Authenticate with your CoreDump account using your private key. The extension securely sends your coding activity to the platform.
  </Step>
  
  <Step title="Track and compete">
    Start coding and watch your stats grow. View your dashboard, climb the leaderboards, and maintain your streak.
  </Step>
</Steps>

<Info>
  CoreDump tracks your coding time automatically based on your active file and language. No manual timers needed.
</Info>

## Public APIs

CoreDump provides public APIs to access your coding statistics programmatically or display them on your portfolio:

- **Stats API**: Get your coding time for today and this week in JSON format
- **Profile API**: Access detailed statistics including languages, achievements, and activity history

Learn more in the [API Reference](/api/overview).

## Built with modern technology

CoreDump is built using industry-standard tools and frameworks:

- **Next.js** with App Router for a fast, scalable web application
- **TypeScript** for type-safe code
- **PostgreSQL** with Prisma ORM for robust data management
- **NextAuth.js** for secure authentication
- **TailwindCSS** for beautiful, responsive UI
